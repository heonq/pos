# POS

## 기능 목록

### 판매 상품 데이터 관리

- 상품 추가

  - [ ] 상품명
  - [ ] 상품 가격
  - [ ] 바코드(선택)
  - [ ] 카테고리

- [ ] 엑셀 파일로 상품 추가

---

- 추가 상품에 대한 유효성 검사

  - [ ] 상품명이 중복될 경우 예외 처리
  - [ ] 바코드가 중복될 경우 예외 처리
  - [ ] 상품 가격이 입력되지 않을 경우 예외 처리
  - [ ] 상품명이 입력되지 않을 경우 예외 처리
  - [ ] 바코드의 경우 입력하지 않아도 등록이 가능

---

- [ ] 상품 수정

  - [ ] 상품명
  - [ ] 상품 가격
  - [ ] 바코드
  - [ ] 카테고리
  - [ ] 판매창 숨기기 / 보이기

- [ ] 상품 수정에 대해서 추가 시와 동일한 유효성 검사

---

- [ ] 상품 삭제
  - [ ] 판매내역이 존재하는 상품은 삭제 불가

---

- [ ] 카테고리 관리
  - [ ] 카테고리 추가/수정
  - [ ] 카테고리 이름
  - [ ] 진열 순서
  - [ ] 보이기 / 숨기기 (카테고리 별 보기 일 때)

---

### 결제

- [ ] 버튼을 통해 결제할 상품을 추가 또는 빼기
- [ ] 바코드 입력을 통해 결제할 상품을 추가

- [ ] 결제 수단 선택

  - [ ] 카드
  - [ ] 현금
  - [ ] 계좌이체
  - [ ] 기타
    - [ ] 기타 사유 입력
    - [ ] 기타 사유를 결제 완료 버튼 누르기 전까지 저장
  - [ ] 분할 결제
    - [ ] 결제 수단과 수단별 금액을 입력

- [ ] 할인 적용
  - [ ] 특정 금액으로 할인 적용
  - [ ] 할인율로 할인 적용
  - [ ] 할인 사유 입력
- [ ] 결제 완료 버튼을 눌러 판매내역 저장
  - [ ] 기타 결제일 경우 기타 사유 저장

### 판매내역

- [ ] 판매 현황 표시

  - [ ] 상품별 판매수량
  - [ ] 결제수단 별 판매금액
  - [ ] 총 판매금액
  - [ ] 준비금

- [ ] 판매내역 수정

  - [ ] 판매내역 삭제
  - [ ] 판매금액 수정
  - [ ] 결제방식 수정
  - [ ] 메모 수정 (할인 사유,기타결제 사유 등)

- [ ] 과거 판매내역 확인

### 현금 점검

- [ ] 화폐 단위 별 수량을 입력
- [ ] 준비금 + 현금결제 금액과 실제 현금 비교
- [ ] 금액 일치 여부를 저장
- [ ] 일치하지 않을 경우 차이 금액 출력

## UI

### 결제창 UI

- [ ] 설정한 보기모드를 서버에 저장한다.
- [ ] 서버에서 보기모드를 불러온다.
- [ ] 불러온 보기모드를 렌더한다.
- [ ] 저장된 보기모드가 없을 경우 전체보기 모드를 렌더한다.
- [ ] 카테고리별 보기를 선택할 경우 카테고리별 보기 모드를 렌더한다.
- [ ] 전체보기 모드를 선택할 경우 전체보기 모드를 렌더한다.
- [ ] 목록을 펼쳤을 때 현재 선택된 보기모드에 아이콘,배경색 등을 통해 표시한다.

#### 카테고리별 보기

<image src="./payment_view1.jpg" width="800px">

- [ ] 서버에서 카테고리 목록을 가져온다.
- [ ] 카테고리를 카테고리 표시 순서에 따라 렌더한다.
- 카테고리 렌더링
  - [ ] 카테고리 이름을 목록 위에 표시한다.
  - [ ] 카테고리 내에 표시 가능한 수량을 넘어가면 가로로 스크롤되도록 한다.
- [ ] 상품 목록에서 표시 가능한 카테고리 수량을 넘어가면 세로로 스크롤되도록 한다.
- [ ] 서버에서 상품 목록을 가져온다.
- [ ] 각 카테고리에 해당하는 상품을 렌더한다.
